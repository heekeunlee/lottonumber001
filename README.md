# 로또9단 1등분석 기법 — 분석 대시보드

도서 **『로또9단 1등 분석기법』(이승윤 · 푸른e미디어)**에서 말하는 분석 기법을 정리하고,
**"왜 당첨자를 많이 배출했다고 하는지"**를 통계적으로 해부하는 단일 페이지 대시보드입니다.

🔗 **배포 URL:** https://heekeunlee.github.io/lottonumber001/

## 다루는 내용

1. **책 개요** — 저자, 구성(6 PART), 핵심 수치
2. **분석 기법 상세** — 제외수 · 고정수 · 미출 기간표(핫/콜드) · 홀짝 · 총합 · 끝수합 · 연번 · 구간분포 · 1등 조합기법
3. **통계 차트** — 홀짝 분포, 총합 분포, 연번 개수, 등수별 확률(로그 스케일)
4. **"당첨자 다수 배출"의 진실** — 대수의 법칙, 제외수의 구조적 트릭, 생존자 편향, 사후 확증 편향
5. **인터랙티브 계산기** — 구독자 규모·게임 수를 바꿔 3등/2등 기댓값을 직접 확인
6. **사실 vs. 주장 비교표**와 결론

## 핵심 메시지

> 통계 패턴은 **진짜**지만 예측력은 **없고**, "당첨자 다수 배출"은 기법의 효과가 아니라
> **모집단 크기와 편향(생존자·사후 확증)**으로 설명된다. 로또는 매 회차 독립 시행이다.
