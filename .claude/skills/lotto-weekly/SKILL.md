# lotto-weekly — 주간 로또 예측 카톡 발송

다가오는 회차의 "예상번호"를 결정론적으로 생성해 카카오톡 나챗방(나에게 보내기)으로 전송한다.
이 예측의 당첨 확률은 무작위와 동일(1/8,145,060)하며, 본 스킬의 목적은 **예측의 무의미함을 사전등록 방식으로 시연**하는 것이다.

## 실행 절차

1. **예측 생성 + 잠금**
   ```bash
   python3 analysis/weekly_predict.py
   ```
   - 출력에서 `<<<KAKAO>>>` 다음 줄부터 끝까지가 **카톡 메시지 본문**이다(≤200자).
   - `analysis/predictions_ledger.json` 에 해당 회차가 `status:"pending"` 으로 잠긴다(이미 있으면 보존).

2. **카카오톡 발송**
   - `<<<KAKAO>>>` 이후 본문을 그대로 `KakaotalkChat-MemoChat` 도구의 `message` 인자로 전달해 발송한다.
   - 200자를 초과하면 발송하지 말고 메시지를 줄여 재시도한다.

3. **원장 커밋** (git 저장소일 때)
   ```bash
   git add analysis/predictions_ledger.json && \
   git commit -m "chore: <회차>회 예상번호 잠금(사전등록)" && git push
   ```
   - 추첨 **전** 커밋해야 사전등록(타임스탬프) 효력이 생긴다.

## 채점(추첨 후, 선택)

토요일 추첨 후 실제 당첨번호로 채점하려면:
```bash
python3 analysis/score_ledger.py   # 실제번호 입력/fetch 후 matches·best 갱신
```
그 결과를 추적 대시보드(`tracker.html`)가 자동 반영한다.

## 주의
- 발송은 외부로 나가는 행위다. 수동 실행 시에는 본문을 사용자에게 먼저 보여주고 발송한다.
- 자동 루틴(매주 월요일)에서는 위 1→2→3을 무인 수행한다.
- 추첨일은 토요일이므로 월요일 발송은 "그 주 토요일 회차" 예측이다.
